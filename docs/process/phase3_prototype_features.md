# Phase 3: プロトタイプ開発（追加機能）

対象機能: オーダー支援・写真管理・通知・検索・美容室管理

## オーダー支援機能
- [ ] 前回のオーダー内容の表示
- [ ] 固定テンプレートの実装（美容師への伝え方例文）
- [ ] 参考スタイル画像の保存（Firebase Storage連携）

## 写真管理機能
- [ ] カット履歴登録時のカメラ・アルバムからの写真選択
- [ ] before / after 写真のFirebase Storageへのアップロード
- [ ] 写真一覧画面の実装
- [ ] 写真の削除処理（履歴削除時に連動）

## 通知機能
- [ ] Firebase Cloud Messaging（FCM）のセットアップ
- [ ] 前回カットからの経過期間に応じたローカル通知の実装
- [ ] 通知間隔の設定（1ヶ月・2ヶ月・3ヶ月）
- [ ] 通知のON / OFF 切り替え

## 検索・フィルター機能
- [ ] 美容室名での絞り込み
- [ ] 日付（期間指定）での絞り込み
- [ ] 満足度での絞り込み
- [ ] 担当美容師名での絞り込み

## 美容室管理機能
- [ ] 美容室一覧画面の実装
- [ ] 美容室登録・編集・削除の実装
  - [ ] 美容室名・住所・電話番号・メモ
- [ ] Firestoreへの保存・取得・更新・削除処理
- [ ] カット履歴登録時の美容室選択連携
