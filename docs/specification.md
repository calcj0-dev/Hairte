# HAIRTE 仕様書

## 1. アプリ概要

- **アプリ名**: HAIRTE（ヘアルテ）
- **名前の由来**: HAIR（ヘアー）+ カルテ
- **目的**: 美容室でのオーダーに困るユーザーのために、髪のカルテ（カット履歴・髪の特徴）を記録・管理し、次回のオーダーをスムーズにする
- **対象ユーザー**: 美容室に通う一般ユーザー（将来的には美容師側にも展開予定）
- **プラットフォーム**: スマートフォンアプリ（iOS / Android）

---

## 2. 機能要件

### 2.1 主要機能

| 機能名 | 概要 | 優先度 |
|--------|------|--------|
| 髪カルテ | 自分の髪の特徴を記録・管理する | 高 |
| カット履歴 | 過去の美容室訪問履歴を記録する | 高 |
| オーダー支援 | 次回のオーダー内容を考えるサポートをする | 高 |
| 通知機能 | 定期的なリマインダーを送る | 高 |
| 検索・フィルター | 履歴を条件で絞り込む | 高 |
| 写真管理 | before / after の写真を保存・閲覧する | 中 |
| 美容室管理 | よく行く美容室・担当美容師を登録する | 中 |
| SNSシェア | ヘアスタイルをSNSにシェアする | 中 |

### 2.2 機能詳細

#### 機能1: 髪カルテ
- 説明: 自分の髪の特徴を項目ごとに記録しておく機能
- 記録項目:
  - 髪質（硬い / 普通 / 柔らかい）
  - 髪の太さ（太い / 普通 / 細い）
  - 髪の量（多い / 普通 / 少ない）
  - くせの強さ（直毛 / 少しくせ / くせ強め）
  - ダメージ具合（健康 / 少しダメージ / ダメージあり）
  - 頭の形・生え癖（自由記述）
  - その他メモ

#### 機能2: カット履歴
- 説明: 美容室に行くたびに記録を残す機能
- 記録項目:
  - 日付
  - 美容室名
  - 担当美容師名
  - オーダー内容（メニュー・スタイル）
  - 仕上がりの満足度（5段階）
  - before / after 写真
  - メモ（気になったこと・次回への引き継ぎ）

#### 機能3: オーダー支援
- 説明: 次回の美容室訪問で何をオーダーするか考えるサポート機能
- 内容:
  - 過去の履歴から「前回のオーダー」を確認
  - 参考にしたいスタイル画像の保存
  - 美容師への伝え方のテンプレート表示（例: 「〇cmカットで、サイドはすっきりさせたい」など）

#### 機能4: 写真管理
- 説明: ヘアスタイルの写真を一覧で確認できる機能
- 内容:
  - カット履歴に紐づいた before / after 写真
  - 参考にしたいスタイル画像の保存

#### 機能5: 通知機能
- 説明: 美容室に行くタイミングをリマインドする機能
- 内容:
  - 前回のカットから指定した期間が経過したら通知（例: 1ヶ月・2ヶ月・3ヶ月）
  - 通知のON / OFF・間隔はユーザーが設定可能

#### 機能6: 検索・フィルター
- 説明: カット履歴を条件で絞り込む機能
- 検索条件:
  - 美容室名
  - 日付（期間指定）
  - 満足度
  - 担当美容師名

#### 機能7: SNSシェア
- 説明: ヘアスタイルの写真やオーダー内容をSNSにシェアする機能
- 対応SNS: Instagram、X（Twitter）、LINE など
- シェア内容: after写真 ＋ オーダー内容のテキスト

#### 機能8: 美容室管理
- 説明: よく行く美容室・担当美容師を登録する機能
- 記録項目:
  - 美容室名
  - 住所 / 電話番号
  - 担当美容師名
  - メモ

---

## 3. 非機能要件

- **パフォーマンス**: 画面遷移は1秒以内
- **セキュリティ**: ユーザーデータはアカウントに紐づけて管理、他ユーザーからアクセス不可
- **可用性**: 99%以上の稼働率
- **スケーラビリティ**: 将来的な美容師側機能の追加を考慮した設計

## 4. 画面設計

### 4.1 画面一覧

| 画面名 | 概要 | 遷移元 |
|--------|------|--------|
| ホーム | 直近の履歴・次回オーダー候補を表示 | 起動時 |
| 髪カルテ | 自分の髪の特徴を記録・編集 | ホーム |
| カット履歴一覧 | 過去の訪問履歴を一覧表示 | ホーム |
| カット履歴詳細 | 個別の訪問履歴を表示・編集 | カット履歴一覧 |
| カット履歴登録 | 新しい訪問履歴を登録 | カット履歴一覧 |
| オーダー支援 | 次回のオーダーを考える | ホーム |
| 美容室一覧 | 登録済みの美容室を一覧表示 | ホーム |
| 設定 | アカウント・通知設定など | ホーム |

### 4.2 画面遷移

```
起動
 └── ホーム
      ├── 髪カルテ
      ├── カット履歴一覧
      │    ├── カット履歴詳細
      │    └── カット履歴登録
      ├── オーダー支援
      ├── 美容室一覧
      └── 設定
```

## 6. 技術スタック

- **フロントエンド**: Flutter（Dart）
- **バックエンド**: Firebase
- **データベース**: Firestore

- **ストレージ**: Firebase Storage（写真保存用）
- **通知**: Firebase Cloud Messaging（FCM）
- **リリース**: App Store（iOS）/ Google Play（Android）
