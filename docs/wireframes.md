凡例: `[ ]` ボタン / `( )` 入力欄 / `[ v ]` セレクト / `★` 評価 / `🖼` 画像

---

## 共通: BottomNavigationBar

髪カルテ・カット履歴をメイン機能として配置。オーダー支援は独立タブを持たずカット履歴に統合。

```
┌─────────────────────────────┐
│  🏠ホーム  📋カルテ  📅履歴  ⚙設定  │
└─────────────────────────────┘
```

## 6. ~~オーダー支援画面~~（削除）

独立した画面は廃止。以下の形でカット履歴に統合済み。

- **テンプレート** → カット履歴登録画面の「テンプレートから入力」ボタン（ボトムシート）
- **参考スタイル画像** → カット履歴詳細画面の「参考スタイル画像」セクション
